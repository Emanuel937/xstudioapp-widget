== Description ==
X-StudioApp Widget for Elementor provides custom widgets designed to extend the Elementor page builder.

The plugin focuses on clean code, performance, and flexibility, allowing users to build modern layouts easily.

Elementor (free version) must be installed and activated.

== Installation ==
1. Upload the plugin to `/wp-content/plugins/`
2. Activate it from the WordPress admin panel
3. Open Elementor and find **X-StudioApp** widgets in the editor

== Frequently Asked Questions ==
= Does this plugin require Elementor? =
Yes, Elementor must be installed and activated.

= Does it work with Elementor Pro? =
Yes, it works alongside Elementor Pro, but it does not replace or unlock Pro features.
